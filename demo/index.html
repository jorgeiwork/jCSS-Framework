Today is:

jCSS Framework HTML5, CSS3, 960gs, Baseline Type!

We are now mobile! :)

a simple, flexible and easy framework

I decided to put together this framework because I need it a base of styles to build websites. How many time have you work on a site and forgotten to style simple HTML tags like <abbr>, <del> or <ins> tags? I must admit there are so many HTML tags that I just can't remember them all! but with this framework, I have pre built "Base" CSS File with all common HTML tags styles, but just in case I forget they are pre style for me.

"Never miss another HTML tag to style, they are all here for you. and if you forget they are already style for you"

I know that there is no such thing as one frame fits all, but even if you don't use this for yourself feel free to take the files and use them to create your own framework, it will improve your productivity and make it easier for you, at least you don't have to type all this CSS, I have already spend the crazy amount of hours doing that for you! :)

I included a lot of extras styles on the typography css such as, over 40 icons with classes to use on <p>,<span> and more. Please check the Typography page for all the samples. I created all of this classes with minimal style because this is meant to be style according to your design concept. What this framework will let you do is not have to type all of those tags and classes from scratch.

This framework is HTML5 Ready there is no reason why not start building for the future, with the help of HTML Shiv even IE6 can undertand HTML5 Tags! Using the 960gs Fluid layout and some extra tweecking now it is very responsive to screensizes, you can test it by simply resizing this window. Think of this as the essesntials to the 960 Grid System, this allowes it to become a full framework, then just a layout system.

"Think of this as the essesntials to the 960 Grid System"

Thank you for stopping by the site, I will be adding more features to this in the future, hope you enjoy and feel free to message me with questions or report bugs at jorge@jorgeiraheta.com
